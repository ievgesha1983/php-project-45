### Brain Games include 5 games
* Even Game. Is it an even number or not?
* Calc Game. Calculate the expression.
* GCD Game. Find the greatest common divisor of given numbers
* Progression Game. What number is missing in the progression?
* Prime Game. Is it a prime number?

### Prerequisites
* Linux, Macos, WSL
* PHP >=8.2
* Git
* Make
* PHP Command Line Tools

#### Install Brain-Games.

[![asciicast](https://asciinema.org/a/N7WwJiDBb4VkaBO8.svg)](https://asciinema.org/a/N7WwJiDBb4VkaBO8)

```

cd php-project-45

make install
```
